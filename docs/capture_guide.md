# Capture Guide

The quality of your clone will rise or fall on the capture, not the glue code.

## Voice reference

For a first-pass clone:

- Record `3` to `5` clean clips
- Keep each clip between `8` and `20` seconds
- Use the same mic and room across clips
- End each take with about `1` second of silence

For a better personal model later:

- Capture `20` to `60` minutes total
- Mix calm, energetic, slow, and punchy delivery
- Keep background noise, music, and reverb out

Recommended format:

- `wav`
- `48 kHz` or `44.1 kHz`
- mono is fine

Important:

- Save the exact transcript for each clip
- Avoid heavy compression, phone speaker playback, and clipped peaks

## Portrait image

For the first talking-head result:

- One front-facing image
- Head and shoulders visible
- Neutral expression
- Good lighting
- No sunglasses, heavy occlusion, or hair covering half the face

Recommended:

- `1024x1024` or larger
- centered face
- clean background

## Driving video for LivePortrait

Use this only after the MVP is working.

- `10` to `20` second clips
- Start with a neutral frontal frame
- Keep the head as the main subject
- Limit large shoulder swings
- Crop near square if possible

Good motion library ideas:

- calm conversational
- energetic emphasis
- slight smile
- thoughtful pause
- strong nodding

## Naming convention

You will save yourself pain if you stay organized.

Examples:

- `data/voice_refs/me_ref_01.wav`
- `data/voice_refs/me_ref_01.txt`
- `data/portraits/me_front.png`
- `data/driving_videos/me_neutral_driver.mp4`

## Ground rules

- Only use footage and voice data you own and consent to use.
- Treat this as a personal media project unless we do a stricter license and compliance pass.
